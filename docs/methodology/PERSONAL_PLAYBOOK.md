# ADHD Productivity Strategies for Freelance Development

Personal playbook for Augustin. Update this as you learn what works.

## Daily Structure

### Morning Startup (10-15 min max)
1. Open Super Productivity
2. Review yesterday's unfinished tasks
3. Pick ONE "rock" task - the most important thing today
4. Schedule 2-3 other tasks max (don't overplan)
5. Start the timer on your rock task

### The "Chaotic Day" Protocol
When you have a variable/chaotic day:
- Use **Flowtime mode** instead of rigid Pomodoro
- Track time passively (just let the timer run)
- Tag tasks with energy level: `#low-energy`, `#high-energy`
- Keep a `#quick-win` backlog for low-motivation moments

### End of Day (5 min)
- Stop all timers
- Review daily summary (SP shows this automatically)
- Move unfinished tasks to tomorrow or backlog
- Note any patterns in the summary

## Task Management

### Project Structure
```
Client A/
  └── Active sprint tasks
Client B/
  └── Active sprint tasks
Admin/
  └── Invoicing, emails, contracts
Personal/
  └── Learning, side projects
```

### Tag System
| Tag | When to use |
|-----|-------------|
| `#quick-win` | <15 min, easy, good for low energy |
| `#deep-work` | Needs focus, schedule when sharp |
| `#blocked` | Waiting on someone else |
| `#admin` | Non-coding work |
| `#bug` | Bug fixes |
| `#feature` | New features |

### Task Sizing
- Break tasks into chunks under 2 hours
- If you estimate >2h, it needs subtasks
- Use SP's time estimation feature to track accuracy

## Focus Strategies

### Starting is the Hardest Part
When you can't start:
1. Set timer for just 5 minutes
2. Tell yourself "I'll just look at the code"
3. Open the file, read for 5 min
4. Often momentum kicks in

### The Procrastination-Buster
SP has a built-in procrastination plugin. When stuck:
1. Click the procrastination button
2. It helps identify WHY you're stuck:
   - Task unclear? → Break it down
   - Task boring? → Pair with podcast/music
   - Task scary? → Start with the easy part
   - Perfectionism? → Set a "good enough" bar

### Hyperfocus Protection
Set break reminders at 45-60 min intervals:
- When reminder pops: stand up, stretch, hydrate
- Don't skip breaks even if in flow
- Breaks prevent burnout and errors

## Time Tracking for Freelancers

### Why Track Time
- Invoice clients accurately
- Learn your actual pace (vs estimates)
- Identify time sinks
- Prove your worth to yourself on bad days

### GitHub Integration
1. Import issues from client repos
2. Start timer when working on issue
3. SP tracks time per issue
4. Export for invoicing

### Weekly Review
Every Friday (15-30 min):
1. Open SP's weekly summary
2. Note total hours per project
3. Check: were estimates accurate?
4. Adjust future estimates based on data

## Emergency Protocols

### "I Did Nothing Today" Recovery
1. Open SP, look at ANY tracked time
2. Even 30 minutes counts - you showed up
3. Identify what blocked you (energy? clarity? motivation?)
4. Set ONE small task for tomorrow morning
5. Be kind to yourself - ADHD is a disability, not laziness

### Overwhelm Protocol
When everything feels like too much:
1. Close all browser tabs except SP
2. Pick the SMALLEST task visible
3. Set timer for 15 minutes
4. Do only that one thing
5. After 15 min: break, then reassess

### Client Deadline Panic
1. List ALL deliverables (in SP)
2. Estimate each honestly
3. If impossible: communicate NOW
4. Better to renegotiate than ghost
