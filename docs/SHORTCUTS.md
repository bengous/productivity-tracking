# ⌨️ Super Productivity Keyboard Shortcuts

Master these shortcuts to operate at the speed of thought.

## 📝 Task Management

| Key | Action |
| :--- | :--- |
| **`Shift + A`** | **Add a new task** (Focus input) |
| **`Enter`** | Edit title of currently focused task |
| **`D`** | Mark task as **Done** |
| **`Backspace`** | **Delete** task |
| **`Shift + Alt + A`** | Add a **Sub-task** |
| **`Space`** | Expand/Collapse task details |
| **`Esc`** | Close overlays / Blur focus |
| **`Arrow Keys`** | Navigate task list |
| **`Ctrl + Shift + ↑/↓`** | Move task up/down |

## ⏱️ Time Tracking

| Key | Action |
| :--- | :--- |
| **`Shift + X`** | **Start/Stop** time tracking for selected task |
| **`Y`** | Toggle time tracking for focused task |

## 🔍 Navigation & Views

| Key | Action |
| :--- | :--- |
| **`Arrow Right`** | Open additional info panel |
| **`S`** | Open **Schedule** dialog |

> **Note:** You can customize these shortcuts in `Settings` -> `Keyboard Shortcuts`.
